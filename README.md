# Projet Une base de donnée World(solo)

Ce petit projet concerne la mise en oeuvre de notre connaissance sur les bases de données et l'application du petit notion en language Sql précisement sur les commandes CRUD. Elle consiste à reproduire un tableau donnée en language sql dans un base de donnée puis le modifier suivant les instructions  et enfin l'envoyer dans le github.

## Pré-requis
Le projet a été crée dans un éditeur texte puis tester dans un server local(localhost) qui est le phpmyAdmin.
